# somproject
using the self organizing map for clustering of text documents
# Requirements
Qt 5.3, cmake, g++.
# Building

```
$ wget https://github.com/nourredinealiane/somproject.git
$ unzip somproject-master.zip
$ cd somproject-master
$ sh execute.sh

```
# Run

```
$ cd build
$ ./som

```
